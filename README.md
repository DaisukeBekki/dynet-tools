# README

### Prerequisite 1: Haskell Stack
In Linux:
```
$ wget -qO- https://get.haskellstack.org/ | sh
```
In Mac:
```
$ brew install haskell-stack
```
See https://docs/haskellstack.org/en/stable/README/#how-to-install for details.

## Prerequisite2: 
In Linux:
```
$ sudo apt-get install nkf gnuplot
```

## Install
```
$ cd <dynet-tools>
$ stack build
```
